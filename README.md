# LemonNEL
一个简易的开源指令提权并免杀，使用`GPL v3`开源协议
# 食用指南
- 1.执行`git clone https://github.com/Colorful-glassblock/LemonNEL.git`
- 2.使用带有 `-ladvapi32 -lshell32 -luser32 -lkernel32 -static` 编译参数的MinGW编译两个cpp文件
- 3.打开`buildcmd.exe` 输入你的指令，就像这样
``` batch
mkdir C:\test
del "C:\test" /f /s /q
```
- 4.按一下`Enter`,如果你是用的是`Windows`请按下`Ctrl+Z`，如果你使用如果你是用的是`类Unix系统`请按下`Ctrl+D`，然后在按下`Enter`
- 5.复制out文件夹到`run.exe`的同目录
- 6.ALL DONE！
